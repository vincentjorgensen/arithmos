 arabic2greek

Rust script that implements the GreekNumeral library in this repository

Example usage
=============

.. code-block:: bash

   cargo build --release

   ./target/release/arabic2greek 1 # Α'
   ./target/release/arabic2greek 616 # ΧΙϜ'
